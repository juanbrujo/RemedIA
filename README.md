# RemedIA 💊

🔒 Repositorio privado.

RemedIA es una aplicación Nuxt 4 para buscar y comparar precios de remedios en las farmacias Farmex y Curie, usando un diseño moderno y responsivo.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```


## Deploy en Vercel

1. Sube este repositorio a GitHub, GitLab o Bitbucket.
2. Ve a [vercel.com](https://vercel.com/) y crea un nuevo proyecto enlazando este repo.
3. Vercel detectará Nuxt automáticamente. No necesitas configuración extra.
4. El build y deploy serán automáticos en cada push.

Para configuración avanzada, revisa `vercel.json` y `.vercelignore`.
